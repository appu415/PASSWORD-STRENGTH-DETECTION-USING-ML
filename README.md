## 🚀 Overview

This project is an **intelligent Password Strength Analyzer** powered by **Machine Learning**.
It evaluates passwords based on structural and statistical features, classifies them into:

* 🔴 Weak
* 🟡 Medium
* 🟢 Strong

Additionally, it estimates **password crack time** and provides **security insights** to improve user awareness.

---

## 🎯 Key Features

✨ Machine Learning-based classification
✨ Feature engineering (length, entropy, character diversity)
✨ Crack time estimation
✨ Multiple ML models (Random Forest, SVM, Logistic Regression)
✨ Real-world cybersecurity use case
✨ Clean and easy-to-understand implementation (Jupyter Notebook)

---

## 🧠 How It Works

```text
User Password
      │
      ▼
Feature Extraction
(length, digits, symbols, entropy)
      │
      ▼
Machine Learning Model
(Random Forest - Best Performer)
      │
      ▼
Strength Prediction
(Weak / Medium / Strong)
      │
      ▼
Crack Time Estimation + Suggestions
```

## 📊 Feature Engineering

The model analyzes:

* Password length
* Number of digits
* Uppercase & lowercase letters
* Special characters
* Entropy (randomness)

---

## 🔐 Crack Time Estimation

Password strength is mapped to estimated crack time:

| Strength | Crack Time      |
| -------- | --------------- |
| Weak     | Seconds         |
| Medium   | Minutes–Hours   |
| Strong   | Years–Centuries |
